Setup

  • 1: you should provide the openAI API key
  • 2: Save it at Options page

To use this extension, you need to provide your OpenAI API key. If you don't have an API key yet, you can sign up for one on the OpenAI website.

You need to right-click on the extension and select "Options" to set the api key.

How to find your OpenAI API Key:

If you already have an OpenAI API key, you can find it on your API keys page. Simply copy and paste it into the input field above and click the "Save API Key" button.
